Making a Project

One thing that I would recommend to any aspiring programmer is that you should always be working on some sort of project if you have some free time. This is not only because it can make you a better programmer, but also because if your project is good enough (such as an app that is published on the app store), then that can be used on your resume to help you get a job. Employers don't always look at just internships, side projects can help boost your resume. Here are some side projects that you can do for fun.

My Ideas

Now I wouldn't recommend using these projects in your resume, but these are just meant for fun and little challenges.

  • Program that orders name from an Excel file
  • Hangman game with GUI
  • Client-server program such as a Slack Clone
  • Twitter Bot
  • Application with a database that can be accessed
